# Changelog

## 0.0.8

New timeOrTask Option for tante start.
All of this is valid
```
    tante start 
    tante start 8:00
    tante start cleanup
    tante start 8:00 cleanup

```
Breaking change, no longer supported:
```

    tante start task 8:00

```
